### スタック · what I build with

TypeScript on Bun, orchestrated through Turborepo. Web is React 19 on Vite with TanStack Router and Query, oRPC, Tailwind and Radix. Mobile is React Native with Expo. Data sits in Postgres behind Drizzle, sessions through Better Auth. AI traffic moves over the Vercel AI SDK across Anthropic, OpenAI, Google, Bedrock and Vertex. Trigger.dev runs the background jobs; E2B and Daytona host the sandboxed code. Chat surfaces reach into Discord, Slack, Telegram, iMessage and Teams.
